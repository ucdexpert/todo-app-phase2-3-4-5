# Todo Application Overview

## Project Description
A full-stack web application for managing todo lists with user authentication, task management, and responsive UI.

## Core Features
- User registration and authentication
- Create, read, update, and delete tasks
- Task filtering by status (all, pending, completed)
- Responsive design for mobile, tablet, and desktop

## Technology Stack
- Frontend: Next.js 16+, TypeScript, Tailwind CSS
- Backend: FastAPI, Python 3.13+
- Database: Neon PostgreSQL with SQLModel ORM
- Authentication: Better Auth with JWT

## Architecture
The application follows a monorepo structure with separate frontend and backend components, communicating through a REST API.
